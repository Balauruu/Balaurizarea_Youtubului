    generation.md -- This is the writer's brain. Must contain ALL of the following sections (per 12-RESEARCH.md Pattern 3):

    1. Role statement: "You are the writer for a dark mysteries documentary channel. You produce narrated chapter scripts from research dossiers."

    2. Input contract: Describe the 3 context files printed by the CLI: Research Dossier (9 sections including Section 4: HOOKs and Section 5: QUOTEs), Style Profile (5 Universal Voice Rules, arc templates, transition library, open ending template), Channel DNA (voice, tone, output targets).

    3. Hook construction: The 4-part formula is ALWAYS followed:
       - Opening quote -- direct speech from authority/witness, no attribution on first line
       - Compressed overview -- location, year, what began, what it became. Two sentences max, no adjectives of scale
       - Misinformation flag -- ONLY when a demonstrable false narrative exists. Omit if not applicable
       - Closing formula -- "This is the true story of [subject] and [what drove it]."

    4. HOOK/QUOTE selection rules:
       - HOOKs are structural anchors -- they determine where chapters begin and what drives them. Select 2-4 from the dossier. The strongest HOOK opens the video.
       - QUOTEs are verbatim spoken words -- always attributed, always introduced by narrator, always set apart. Pattern: narrator introduces speaker, quote speaks, narrator resumes. Example from CONTEXT.md decisions.
       - HOOKs and QUOTEs are NOT the same thing. Do not use a HOOK as verbatim narration or a QUOTE without attribution.

    5. Chapter structure rules:
       - Derive chapter breaks from Research.md content: timeline, hooks, key figures, narrative tension points
       - 4-7 chapters (soft guardrail from channel.md)
       - H2 format: ## N. Evocative Title
       - No predefined arc template required. Universal Voice Rules apply to all topics. Template A (Cult arc) only applies to cult/group radicalization topics.
       - EXPLICITLY STATE: topics that are not cult/group radicalization must not use Template A. Derive chapter structure from the research timeline and narrative tension.

    6. Voice rules reference: Enumerate all 5 Universal Voice Rules by name:
       - Rule 1: Declarative Factual Claims -- No Modal Qualifiers
       - Rule 2: Banned Vocabulary -- No Intensifiers, Superlatives, or Clickbait Language
       - Rule 3: Narration Scope -- No Host Commentary, No Fourth-Wall Breaks
       - Rule 4: Source Attribution Syntax -- Label Speculation Explicitly
       - Rule 5: Sentence Rhythm -- Short Declarative Beats After Heavy Information
       For each rule, include a one-line summary. Tell Claude to read the full rule definitions from the Style Profile section of the context package.

    7. Output format constraints:
       - Script.md starts DIRECTLY with ## 1. [Chapter Title] -- no header, no metadata, no table of contents
       - Each chapter: H2 heading + continuous prose paragraphs
       - NO bullet points, sub-sections, stage directions, visual cues, production notes, host commentary
       - Chapter titles use evocative register (what it feels like), not descriptive (what happens)
       - Per-chapter word count is not enforced -- natural flow determines length
       - Total target: 3,000-7,000 words

    8. Open Ending Template trigger: Apply when the case has contested resolution, permanently incomplete record, or moral weight in not-knowing. Three-part structure: present final known evidence, acknowledge unknowns, leave weight with audience. Do NOT provide artificial resolution or consolation.

    9. Output instruction: "Write Script.md to the output path shown at the end of the context package. Use the Write tool. Do not print the script to stdout."
