## Out of Scope

- Upload scheduling or publishing automation
- Real-time monitoring or notifications
- Social media analysis (Reddit, Twitter/X)
- Third-party analytics integration (SocialBlade, VidIQ)
- LLM API wrappers for reasoning
- PDF extraction (PyMuPDF/OCR)
- Paid source access (PACER, newspaper archives)
- NLP/spaCy style metrics
- Inline visual/production notes in script
- Automated fact-checking against external sources
- Word count gates or automated quality scoring
