<feature>
  <name>Topic Generation Helper Functions (topics.py)</name>
  <files>.claude/skills/channel-assistant/scripts/channel_assistant/topics.py, tests/test_channel_assistant/test_topics.py</files>
  <behavior>
    **_load_past_topics(path: Path) -> list[str]:**
    - Empty/missing file -> returns []
    - File with "- **Title One** | 2026-01-01 | summary" lines -> returns ["Title One"]
    - File with mixed content -> extracts only bold-marked titles
    - Handles both "- **Title**" and "**Title**" prefix formats

    **check_duplicates(candidate_title: str, past_titles: list[str], threshold: float = 0.85) -> str | None:**
    - Exact match -> returns matching past title
    - Near-match above threshold (e.g., "The Matamoros Murders" vs "The Matamoros Cult Murders") -> returns match
    - Distinct topic (e.g., "Flight 980" vs "The Matamoros Murders") -> returns None
    - Case insensitive comparison
    - Empty past_titles list -> returns None

    **load_topic_inputs(root: Path) -> dict:**
    - Returns dict with keys: "analysis" (str), "channel_dna" (str), "past_topics" (list[str])
    - Reads from context/competitors/analysis.md, context/channel/channel.md, context/channel/past_topics.md
    - Missing files: analysis and channel_dna raise FileNotFoundError, past_topics returns []

    **write_topic_briefs(briefs: list[dict], output_path: Path) -> None:**
    - Creates parent directories if they don't exist
    - Writes markdown with header "# Topic Briefs" and generation timestamp
    - Each brief includes: numbered heading, pillar, score line (O:X C:X S:X V:X = N/20), estimated runtime, hook, timeline entries, scoring justification for each dimension
    - Overwrites file on each call

    **format_chat_cards(briefs: list[dict]) -> str:**
    - Returns compact multi-line string with numbered cards
    - Each card: [N] Title -- hook (1 line)
    -           Scores line: O:X C:X S:X V:X = N/20 | ~Xmin | Pillar
    -           Optional: duplicate warning tag, underserved cluster tag
  </behavior>
  <implementation>
    Implement all functions in topics.py using only stdlib (pathlib, re, difflib, datetime).

    past_topics.md canonical line format: `- **Title** | YYYY-MM-DD | one-line summary`

    Topic brief dict schema (used by write_topic_briefs and format_chat_cards):
    ```python
    {
        "title": str,
        "pillar": str,  # Content pillar label
        "hook": str,  # 1-2 sentence hook
        "timeline": list[str],  # 3-8 timeline entries
        "scores": {
            "obscurity": int,  # 1-5
            "complexity": int,  # 1-5
            "shock_factor": int,  # 1-5
            "verifiability": int,  # 1-5
        },
        "justification": {
            "obscurity": str,
            "complexity": str,
            "shock_factor": str,
            "verifiability": str,
        },
        "estimated_runtime_min": int,
        "duplicate_of": str | None,  # Past topic title if near-duplicate
        "tags": list[str],  # e.g., ["UNDERSERVED CLUSTER: True Crime & Cults"]
    }
    ```
  </implementation>
</feature>
