    1. `scrape_autocomplete(keyword: str) -> list[str]` — Uses urllib.request to GET `https://clients1.google.com/complete/search?client=youtube&hl=en&q={keyword}`.     2. `scrape_search_results(keyword: str, max_results: int = 20) -> list[dict]` — Uses crawl4ai AsyncWebCrawler to fetch YouTube search page. Extracts ytInitialData via regex from HTML. Navigates JSON path: contents > twoColumnSearchResultsRenderer > primaryContents > sectionListRenderer > contents[N] > itemSectionRenderer > contents[] > videoRenderer. Returns list of dicts: {video_id, title, channel, published_text, views_text}. Wrapped in `_run_async()` helper using `asyncio.run()`. Full try/except around JSON traversal — returns empty list on any KeyError/IndexError.

    3. `get_recent_competitor_videos(db: Database, days: int = 30) -> list[dict]` — Queries SQLite: `SELECT v.title, v.upload_date, c.name AS channel_name FROM videos v JOIN channels c ON v.channel_id = c.youtube_id WHERE v.upload_date IS NOT NULL AND v.upload_date >= ? ORDER BY v.upload_date DESC`. Returns list of dicts.

    4. `derive_keywords(channel_dna: str, topic_clusters: list[str]) -> list[str]` — Parses channel DNA for content pillars section, extracts keywords. Combines with topic_clusters (up to 5 each). Falls back to CONTENT_PILLAR_KEYWORDS constant if parsing fails. Deduplicates.
